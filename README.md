# Автоматизированный инструмент мониторинга угроз (Threat Monitoring Tool)

Данный проект выполнен в рамках итогового домашнего задания по дисциплине «Программирование на Python». Скрипт предназначен для автоматизированного сбора, анализа и визуализации данных по кибербезопасности. В случае недоступности Апи скрипт использует Демо-данные. В данном скрипте вписаны апи-ключи от VT и Vulners используемые в рамках демо-программ.

В рамках отработки запроса не удалось добиться ответа от Vulners, вероятно ошибка 403 следствие работы cloudflare, т.к. апи активны. Поэтому отчет составляется по тестовым данным.

## Функциональные возможности

Скрипт реализует полный цикл обработки угроз:
1. **Сбор данных из 3-х источников:**
   **VirusTotal API**: Проверка репутации IP-адресов.
   **Vulners API**: Поиск актуальных уязвимостей по заданным критериям.
   **Suricata Logs**: Анализ имитированных сетевых логов на наличие сигнатур атак.
2. **Анализ и фильтрация:**
   Автоматический поиск критических уязвимостей с баллом CVSS >= 7.0.
   Выявление подозрительной сетевой активности и вредоносных IP.
3. **Система реагирования:**
   Вывод уведомлений о критических угрозах в консоль.
   Имитация автоматической блокировки портов/IP при обнаружении сетевых атак.
4. **Отчетность и визуализация:**
   Генерация итогового отчета в формате `security_report.json`.
   Построение графика распределения баллов CVSS в формате `vulnerability_graph.png`.

## Технологический стек
- **Язык**: Python 3.8+
- **Библиотеки**: `requests` (API), `pandas` (обработка данных), `matplotlib` (визуализация), `json` (отчетность).
